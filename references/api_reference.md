# 风格评分参考

## 0-100 评分锚点

- 0-20：该特征几乎不存在。
- 21-40：弱存在，仅局部可见。
- 41-60：中等存在，整体可感知。
- 61-80：明显存在，成为主要视觉特征。
- 81-100：强主导，几乎定义画面风格。

## 全局维度判定

- 线稿定义度：看轮廓和结构是否主要依赖线条边界建立。
- 体积塑形强度：看明暗建模、法线感、次表面/高光等是否塑造体积。
- 写实度：看比例、材质、光照、镜头是否接近真实世界视觉规律。
- 风格化夸张度：看形体变形、色彩偏离现实、符号化处理是否明显。
- 边缘硬度：看主体边缘是否干脆、剪影是否锋利。
- 饱和度：看高纯色面积与平均彩度。
- 对比度：看明暗跨度与局部反差强度。
- 冷暖倾向：按主导光色与环境反射综合判断。

## 角色维度判定

- 角色结构写实度：看五官比例、骨骼肌肉逻辑、服装重力是否合理。
- 材质细节密度：看皮肤、织物、金属、头发等微细节与纹理层级。
- 角色相对环境优势：优先判断清晰度，其次亮度与饱和度，再看细节密度。
- 细节重点：在脸 / 眼 / 头发 / 服饰材质中选最显著两项。

## 环境维度判定

- 结构主导 / 氛围主导：前者重透视和形体，后者重光雾、色温、情绪。
- 背景信息量：
  - 低：大色块+少量元素；
  - 中：有层次与若干可识别物件；
  - 高：多层细节、纹理和叙事元素。
- 空气感：可用“通透/薄雾/潮湿/尘感/夜雾”等。
- 主色调：优先写“色相+明度感”，如“低明度冷蓝灰”。
- 天气/时间：晴、阴、雨、雪；晨、午、暮、夜。
- 光源：主光方向+光质（硬/软）+是否有辅光/轮廓光。

## 风格介质判定

- 2D手绘倾向：看线条感、笔触感、块面化简与非真实材质表达。
- 摄影写实倾向：看镜头景深、噪点/胶片特征、真实光学与曝光逻辑。
- 3D/CG倾向：看几何体积一致性、材质BRDF规律、渲染高光/反射特征。
- NPR（非真实感渲染）倾向：看是否使用卡通渲染、描边、分层阴影、风格化法线。
- 纹理颗粒强度：看胶片颗粒、画布肌理、后期噪声是否显著。
- 说明：以上维度可并存，不强制互斥；按视觉主导程度分别打分。

### 介质锚点示例（便于统一尺度）

- 2D手绘倾向 80+：明显线稿/笔触主导，真实材质让位于画风表达。
- 摄影写实倾向 80+：接近真实摄影静帧，曝光与镜头语言可信。
- 3D/CG倾向 80+：材质响应与几何体积呈现出典型数字渲染特征。
- NPR 倾向 80+：可见强非真实渲染语言（如描边、赛璐璐分层阴影）。

## 关系与统一度判定

- 分离手段：明度差、饱和差、冷暖差、边缘清晰度差。
- 统一度高（70+）：光源逻辑一致、材质响应一致、色彩系统同源。
- 统一度低（<50）：前后景像拼贴、受光方向冲突、材质噪声不一致。

## 排除项推荐库（优先挑 3 条）

- 避免过曝高光
- 避免暗部死黑
- 避免脏灰肤色
- 避免线稿锯齿与断裂
- 避免背景细节抢主体
- 避免光源方向冲突
- 避免冷暖关系混乱
- 避免材质塑料感过强
- 避免后期锐化过度
- 避免角色与背景贴图感
- 避免皮肤蜡像感（Uncanny）
- 避免AO/轮廓阴影过重导致脏边
